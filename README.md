# App Clima

Aplicação em Python que permite obter informações meteorológicas de forma simples, utilizando dados de uma API externa.

## Objetivo do Projeto
O objetivo desta aplicação é:
- Consultar dados meteorológicos (ex: temperatura, estado do tempo)
- Apresentar a informação de forma clara ao utilizador
- Servir como projeto de aprendizagem e portefólio em Python

## Tecnologias Utilizadas
- Python 3
- Requests (para chamadas HTTP)
- API de dados meteorológicos "Open-meteo"
